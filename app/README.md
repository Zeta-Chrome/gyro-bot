Controller App for GyroBot

The controller application for the GyroBot includes the following components:

Joystick widget – Used to control the robot’s movement.

Server control buttons – Provide quick access to essential control functions.

Image canvas – Displays the TCP image stream from the ESP32 camera.

Mode selectors – (Optional; may be removed in the future.)

Info dropdowns – Show real-time data such as joystick output, IMU readings, and ultrasound measurements.

Configuration panel – Allows the user to adjust various configuration settings for the robot.
